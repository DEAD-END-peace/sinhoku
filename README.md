# sinhoku
方位角・距離計算ツール

## 概要
`sinhoku` は、2地点の緯度経度を入力して方位角（方位方向）と距離を計算できるウェブツールです。  
GoogleマップやAppleマップのリンクを自動生成する機能も搭載しています。

## 主な機能
- 緯度経度（度分秒または10進数）入力対応
- 方位角（Azimuth）と距離の計算
- Googleマップ / Appleマップのリンク自動生成
- iPhone/Android対応のリンク切替
- 小数点や括弧付き座標の自動認識

## 使い方
1. [Webページを開く](https://dead-end-peace.github.io/sinhoku/)
2. 出発地点と目的地点の緯度・経度を入力
3. 「計算」ボタンを押すと、方位角と距離が表示されます
4. 必要に応じてマップリンクをクリックして表示
